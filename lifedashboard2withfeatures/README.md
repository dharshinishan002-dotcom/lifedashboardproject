# 🌸 Life Dashboard

> A modern, aesthetic personal productivity web app designed to help users manage goals, daily plans, and habits in one place.

✨ Built with a focus on **clean UI, pastel design, and real-world functionality**

---

## 🚀 Live Demo

🔗 https://yourusername.github.io/life-dashboard/

---

## ✨ Features

### 🎯 Goals Tracker
- Add new goals
- Mark goals as completed (strike-through)
- Delete goals
- Track completion progress

### 📅 Daily Planner
- Add daily tasks
- Mark tasks as done
- Remove tasks easily

### 🧠 Habit Tracker
- Add habits
- Track daily completion using checkboxes
- View habit completion stats
- 🔥 Streak tracking for consistency

### 📊 Dashboard
- View overall progress:
  - Goals completed
  - Plans completed
- Live clock
- Motivational quotes

### 🧾 Profile System (Popup UI)
- Save personal details:
  - Name
  - Phone number
  - Additional info
- Edit profile anytime
- Clear profile data

### 💾 Data Persistence
- Uses Local Storage
- Data stays even after refresh

### 🌙 Dark Mode
- Toggle between light and dark theme

### 📱 Responsive Design
- Works smoothly on:
  - Mobile 📱
  - Tablet 📟
  - Desktop 💻
